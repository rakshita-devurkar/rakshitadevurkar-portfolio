# Rakshita Devurkar Portfolio

This project is a self-contained portfolio website for Rakshita Devurkar, positioned for senior and AI-native software engineering roles.

It includes:

- A high-impact hero section with hiring-oriented positioning
- Interactive portfolio lenses for product, AI systems, and leadership framing
- Skill dashboards for front-end, back-end, LLM infrastructure, AI economics, and responsible AI
- Thought-process sections about how software engineering is changing in an AI-first world
- Project spotlights designed to support hiring conversations with top Bay Area AI companies

## Run

Open [index.html](/Users/saurabhsalunkhe/Documents/New%20project/index.html) directly in a browser, or run:

```bash
npm start
```

Then visit [http://localhost:3000](http://localhost:3000).

## Verify

npm test
